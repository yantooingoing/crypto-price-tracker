# Crypto Price Tracker 🚀

Fetch real-time crypto prices using the free CoinGecko API.
Supports USD and IDR (Indonesian Rupiah).

## Features
- Real-time price in USD & IDR
- Supports BTC, ETH, SOL, and more
- Clean formatted output

## Requirements
- Python 3.x
- requests library

## How to Run
pip install requests
python tracker.py
```
